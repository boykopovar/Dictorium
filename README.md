# Dictorium

Dictorium - это C++ библиотека, посвящённая реализации и исследованию **алгоритмов поиска элементов по ключу**.

Библиотека будет включать различные реализации `Key-Value` структур (наследников [**IDictionary**](include/Dictorium/Contracts/IDictionary.h)).

Реализованные классы и интерфейсы находятся в пространстве имен `dtr`.
