# Соглашения по написанию кода `Dictorium`


### Именования
- Название интерфейса (класса с виртуальными методами без реализации) начинается с `I` (как в C#).
- Названия классов, публичных методов: PascalCase.
- Названия приватных методов и полей: _camelCase.
- Названия параметров шаблона в стиле C#: `TKey`, `TValue`, `TIterator`.

### Структура
- Все сущности библиотеки в пространство имен `dtr`.
- Заголовочные файлы классов в папке `include/Dictorium/Entities`
- `CPP` реализация методов класса в папке `src/Entities` в подпапке с названием класса.
- `md` файл описания класса (наследника `IDictionary`) в папке `docs/Entities`.

`md` файл описания класса должен содержать таблицу:

| Метод            | Сложность | Аморт. сложность | Пояснение |
|------------------|-----------|------------------|-----------|
| _get             |           |                  |           |
| InsertOrAssign   |           |                  |           |
| ContainsKey      |           |                  |           |
| TryGetValue      |           |                  |           |
| Add              |           |                  |           |
| Remove           |           |                  |           |
| Clear            |           |                  |           |
| Count            |           |                  |           |
| Items            |           |                  |           |
| initializer_list |           |                  |           |
