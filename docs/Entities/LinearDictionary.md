# LinearDictionary

`LinearDictionary<TKey, TValue>` - простая реализация словаря поверх `std::vector<std::pair<TKey, TValue>>`

| Метод            | Сложность | Аморт. сложность | Пояснение                                                      |
|------------------|-----------|------------------|----------------------------------------------------------------|
| GetValue         | O(n)      | O(n)             | Линейный поиск, бросает исключение если не найден              |
| InsertOrAssign   | O(n)      | O(n)             | Поиск ключа линейным проходом                                  |
| ContainsKey      | O(n)      | O(n)             | Линейный поиск по `_dict`                                      |
| TryGetValue      | O(n)      | O(n)             | Линейный поиск по `_dict`                                      |
| Add              | O(n)      | O(n)             | `ContainsKey` → O(n) + `emplace_back` → O(1)                   |
| Remove           | O(n)      | O(n)             | Поиск O(n) + `vector::erase` со сдвигом                        |
| Clear            | O(n)      | O(n)             | Уничтожение всех элементов                                     |
| Count            | O(1)      | O(1)             | `vector::size()`                                               |
| Items            | O(1)      | O(1)             | Возвращается ссылка                                            |
| initializer_list | O(n²)     | O(n²)            | Каждый элемент добавляется через `operator[]` → линейный поиск |

Поиск выполняется **линейным проходом по массиву**.
Контейнер предназначен как **пример структуры наследника `IDictionary`**.

---

# Назначение

`LinearDictionary` не оптимизирован для больших данных.
Структура используется как **пример реализации интерфейса словаря**.
