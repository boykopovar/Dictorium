# ChainingHashDictionary

`ChainingHashDictionary<TKey, TValue, Hash>` — реализация словаря поверх `std::vector<std::list<std::pair<TKey, TValue>>>` с разрешением коллизий методом цепочек.

| Метод            | Сложность | Аморт. сложность | Пояснение |
|------------------|-----------|------------------|-----------|
| GetValue         | O(n)      | O(1)             | `_getIndex` → O(1) + линейный перебор корзины → O(1) в среднем; O(n) в худшем (все ключи в одной корзине) |
| InsertOrAssign   | O(n)      | O(1)             | `_getIndex` + перебор корзины → O(1) в среднем; рехэширование добавляет O(1) амортизированно |
| ContainsKey      | O(n)      | O(1)             | `_getIndex` + линейный перебор корзины → O(1) в среднем |
| TryGetValue      | O(n)      | O(1)             | `_getIndex` + линейный перебор корзины → O(1) в среднем |
| Add              | O(n)      | O(1)             | `ContainsKey` → O(1) + `list::emplace_back` → O(1); `_ensureCapacity` → O(1) амортизированно |
| Remove           | O(n)      | O(1)             | `_getIndex` + перебор корзины → O(1) в среднем; `list::erase` по итератору → O(1) |
| Clear            | O(n + m)  | O(n + m)         | Уничтожение всех `n` элементов во всех `m` корзинах + `resize(8)` |
| Count            | O(1)      | O(1)             | Возвращает поле `_count` |
| initializer_list | O(n)      | O(n)             | `n` вызовов `Add` → O(1) каждый; рехэширование не более O(n) суммарно |

> **Худший случай** для операций поиска/вставки/удаления — **O(n)**, когда функция хэширования отображает все ключи в одну корзину. При равномерном распределении хэшей и коэффициенте загрузки ≤ 0.75 средняя длина цепочки стремится к константе.

---

# Назначение

`ChainingHashDictionary` предоставляет **O(1) амортизированный** доступ к элементам при равномерном распределении хэшей.

Рехэширование выполняется при превышении коэффициента загрузки `0.75` и удваивает размер таблицы, что обеспечивает **O(1) амортизированную** стоимость вставки.

Итератор обходит все корзины последовательно за **O(n + m)**, где `m` — текущее число корзин.
